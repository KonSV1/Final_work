# Данная программа из имеющегося массива строк формирует новый массив, содержащий элементы исходного массива, длина которых составляет три и менее символа.
## Алгоритм решения

1. Создаем произвольный массив __"string arrey[ ]"__;
2. Подсчитываем количество элементов массива с количеством символов менее или равным 3-м символам;
3. Создаем новый пустой массив __"string result [ ]__" c количеством элементов равным посчитанному в п. 2;
4. Найти в массиве _"arrey"_ элемент имеющий длину 3 символа и менее;
5. Записать в массив _"result"_ элемент найденный в п. 4;
6. Повторяем шаги 4 и 5 пока не проверим все элементы массива "arrey";
7. Для проверки работы программы выведем на экран оба массива.
